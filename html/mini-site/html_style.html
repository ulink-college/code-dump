Internet Principles of Operation

Information about how the internet and services works for IGCSE Computer Science

Glossary:

HTTP: a set of rules for transferring files across the internet using the World Wide Web to exchange information found on web pages.

HTTPS: means HTTP over Secure - when your data is sent using a 'secure socket layer' (SSL) or some form of encryption.

HTML: Markup language used for presenting information on the Internet

MAC address: Unique address of the network hardware of a device connected to the network. Uses hexadecimal numbering system

Internet Protocol (IP) address: Unique address given to each and every device connected to the network. E.g. 109.108.158.1

Uniform Resource Locator (URL): the address of a resource on the Internet (or other network - Intranet, LAN etc)
